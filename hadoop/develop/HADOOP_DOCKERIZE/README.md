# Hadoop dockerize
- Connect hadoop and HBase
- Mapreduce job done 
    - with data in HBase and put resul in HBase
- HBase
    - Pseudo distributed mode
    - HBase contains input data and output data in the tables

## versions
- base: 
    - ubuntu 20.04
- java JDk
    - javac 1.8.0_342
- hadoop
    - hadoop 3.2.2
    - zookeeper 3.7.1
    - HBase 2.4.14
