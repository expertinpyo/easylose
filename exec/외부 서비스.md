# 외부 서비스 목록

## 소셜 로그인 (OAuth2)

이용자의 이름 및 식별 목적으로 사용

### Google

- https://console.cloud.google.com/
- 활용 scope:
  - profile

### Kakao

- https://developers.kakao.com/
- 활용 scope:
  - profile_nickname
  - profile_image

### Naver

- https://developers.naver.com/main/
- 활용 scope
  - name
  - profile_image
